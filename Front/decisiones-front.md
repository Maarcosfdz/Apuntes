# Decisiones Front

## Vite vs Next.js
Uso Vite cuando:
- proyecto simple
- no necesito SSR

Uso Next.js cuando:
- necesito SEO
- SSR / SSG
- routing avanzado

## Storage en formularios
Uso sessionStorage para:
- formularios multi-step temporales

Uso localStorage para:
- persistencia más larga (drafts)
