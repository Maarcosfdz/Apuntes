# Conceptos Front

## React
Librería para construir interfaces con componentes.

## Vite vs Next.js
- Vite → más simple, rápido para proyectos pequeños/medianos
- Next.js → framework completo (routing, SSR, SSG, etc.)

## LocalStorage vs SessionStorage
- localStorage → persiste aunque cierres el navegador
- sessionStorage → se borra al cerrar pestaña

## Wizard
Formulario dividido en pasos guiados (multi-step form)
